# ⛽ FuelEase LK

A smart fuel management web application for Sri Lanka to help users find fuel stations, check eligibility, and track queue status in real-time.

## 🚀 Features

* QR-based vehicle identification
* Odd/Even fuel eligibility checker
* Real-time fuel station map
* Crowdsourced queue tracking
* Admin dashboard
* Progressive Web App (PWA)

## 🛠 Tech Stack

* React.js + Tailwind CSS
* Node.js + Express
* MongoDB
* Socket.io
* Leaflet (OpenStreetMap)

## 📦 Installation

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## 🌍 Environment Variables

Create a `.env` file in server:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```
